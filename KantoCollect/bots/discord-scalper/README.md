## Planned Features

- Extract product info from messages
- Look up prices via PriceCharting
- Determine if deal meets criteria

### Auto-Purchase
- Automated checkout on supported platforms
- Configurable buy rules
- Spending limits

### Safety Features
- Kill switch for all auto-purchases
- Purchase confirmations (optional)
- Spending caps per hour/day
- Blacklist/whitelist products

## Setup (Future)

1. Create Discord bot at https://discord.com/developers
2. Add bot to your server
3. Configure watched channels
4. Set up buy rules
5. Enable auto-purchase
